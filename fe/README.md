# Hyper-AIgent Frontend

Frontend service for **Hyper-AIgent**, an AI-powered data interaction platform that allows users to upload documents and connect external data sources such as spreadsheets to interact with them using natural language.

This repository contains the **Next.js frontend** responsible for the user interface and interaction with the backend API.

---

## Overview

Hyper-AIgent frontend enables users to:

- Upload **PDF, CSV, and Excel files**
- Connect **Google Sheets**
- Store and manage documents
- Process documents for AI interaction
- Communicate with AI via conversational APIs

The frontend integrates with **Next.js** for the user interface and **OpenAI APIs** for AI responses and can optionally process large documents via **AWS Lambda pipelines**.

## Run the frontend

```
npm run dev
```
